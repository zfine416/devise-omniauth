== README

Basic application with standard Login/Registration using Devise and Facebook omniauth. Additionally omniauth (Google, Twitter, Github or ect) can easily be added. 
Setup:
* Create a local_env file
* Add newly created file to .gitignore, so it is not tracked
* Add necessary environment vairiables to local_env file
* Facebook ex:
* FACEBOOK_KEY: "123456789"
# FACEBOOK_SECRET: "987654321"
<tt>rake doc:app</tt>.
